# **App Name**: AkilDoc

## Core Features:

- User Authentication: Secure user login/registration using Firebase Authentication.
- Document Upload: Allow users to upload PDF or TXT documents, storing them in Firebase Storage.
- Document Listing: Display uploaded documents in a card-based list.
- Keyword Search: Enable keyword-based search across documents.
- Semantic Question Answering: Use a large language model to answer questions based on document content, citing the source documents. LLM is a tool to augment knowledge with reasoning.
- Document Summarization: Automatically generate short summaries for each document and longer, more detailed ones if the user asks.
- AI Error Reporting: Implement a system for users to report and flag incorrect AI-generated responses, enabling continuous model feedback and improvement. Include a intentionally incorrect response as part of a demo.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and intelligence.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Yellow (#FFD54F) to highlight important information and CTAs.
- Body and headline font: 'Inter' (sans-serif) for readability and a modern feel.
- Simple, consistent icons for document types and actions.
- Card-based layout for document listing with clear visual hierarchy.
- Subtle transitions and animations for a smooth user experience.
